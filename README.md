# Scoreboard App 

A simple and interactive scoreboard application built using JavaScript to track and update scores dynamically.

## Features

* Add and update scores in real-time
* Interactive UI with dynamic updates
* Simple and responsive design


## Tech Stack

* HTML
* CSS
* JavaScript


## Live Demo

https://imaginative-peony-d4be6e.netlify.app/


## Project Structure

* `index.html` – Structure of the app
* `style.css` – Styling
* `script.js` – Logic for updating scores
